# Portfólio de Desenvolvedor Frontend

Bem-vindo ao meu portfólio de desenvolvedor frontend! Este repositório contém os arquivos HTML, CSS e JavaScript que compõem meu portfólio pessoal.

## Visão Geral

Meu portfólio tem como objetivo mostrar minhas habilidades e projetos como desenvolvedor frontend.
O projeto foi estruturado para se parecer com a estrutura do Visual Studio Code da maneira que eu o utilizo no trabalho diariamente, porém como aplicação web que inclui seções como:

- **Sobre Mim:** Uma breve introdução sobre quem sou e minha paixão pelo desenvolvimento web.
- **Projetos:** Uma lista dos projetos em que trabalhei, com links para os respectivos repositórios ou sites ao vivo.
- **Currículo:** Meu currículo profissional completo.
- **Habilidades:** Uma lista das tecnologias que domino, incluindo HTML, CSS, JavaScript, frameworks como Bootstrap, React, entre outras.
- **Contato:** Informações de contato para que possam me encontrar e me contatar.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- `src/`: Pasta contendo toda a estrutura do projeto.
- `html/`: Pasta contendo os arquivos estruturais do projeto em HTML5.
- `css/`: Pasta contendo os arquivos de estilização do projeto em CSS3.
- `js/`: Pasta contendo os arquivos de dinamicidade do projeto em JAVASCRIPT.
- `assets/`: Pasta contendo imagens utilizadas no portfólio.

## Como Visualizar

O projeto ainda não foi feito deploy mas você pode clonar este repositório e abrir o arquivo `index.html` em seu navegador para ver como está indo. Quando o projeto estiver finalizado, incluirei aqui o link para visualização.
